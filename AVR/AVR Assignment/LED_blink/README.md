 
# LED Toggle using AVR Delay

 

 
## Objective
Toggle LED connected to PORTD Pin 5 with 3 seconds ON and 5 seconds OFF timing.
   
## Hardware Used  
- Microcontroller: **ATmega328P**
- LED connected to **PORTD5**  

## Key Concepts Used  
- GPIO manipulation  
- Delay functions  
- Basic embedded loop control

## Workflow 
1. Initialize PD5 as output  
2. Turn LED ON  
3. Delay 3 seconds  
4. Turn LED OFF  
5. Delay 5 seconds  
6. Repeat
